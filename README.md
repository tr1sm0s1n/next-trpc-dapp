## âš™ï¸ Run Locally

Clone the project

```bash
git clone https://github.com/DEMYSTIF/next-trpc-dapp.git
cd next-trpc-dapp
```

Install dependencies

```bash
npm install
```

Test contract

```bash
npm test
```

Install ganache globally

```bash
npm install -g ganache
```

Run ganache on port 8545 (default)

```bash
ganache -i <network_id> -p 8545
```

Deploy contract

```bash
npm run deploy
```

Start the application

```bash
npm run dev
```
