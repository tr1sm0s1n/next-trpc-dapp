Clone the project

```bash
git clone https://github.com/tr1sm0s1n/next-trpc-dapp.git

```

Install dependencies

```bash
npm install
```

Test contract

```bash
npm test
```

Run hardhat node on port 8545

```bash
npm run node
```

Deploy contract

```bash
npm run deploy
```

Start the application

```bash
npm run dev
```
